Intro

Welcome to my ePortfolio, the site consists of four pages. The Intro page, which you are on, outlines the content and structure of the site. The Work page discusses the design and implementation of some projects I have worked on both in and outside of school. The About page details some of my course work at the University of Minnesota-Twin Cities, discusses my career interests, and contains a link to my resume . Finally, the Contact page contains a form to contact me if you are intersted in my services.

Work

C++ Bus-Simulation: This software is an extension of a program written by the University of Minnesota department of computer science. I have added considerable functionality that utilizes many common design patters such as the Observer, Factory, and Decorator patterns. The back-end is written in C++ and makes use of high cohesion/low coupling heuristics. The code can be viewed on my github account, but the simulation must be run on a U of M CSE machine.

GitHub

Emergency room database: For this class project my team and I designed and implemented a database for a hypothetical emergence room. We started with a list of requirements, from which we developed and ER diagram. From there me mapped that diagram into a schema, and finally wrote the SQL scrips to make a functional database.

GitHub

JavaScript Battleship game: This page enables the user to play an online version of the popular board game Battleship. The app was constructed using JavaScript/HTML5/CSS and was adapted from the book Head First JavaScript Programming. It makes use of basic OOP principles and the DOM.

Live version GitHub
